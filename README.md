### Course Learning Goals
Upon completion of this course, students are able to:
- Successfully design, develop, and deploy web-based software using HTML, CSS, PHP, and JavaScript
- Understand how web technologies work and interact with each other
- Implement both client-side and server-side programming solutions
- Integrate databases with web applications
- Deploy and configure web servers with SSL/TLS security

### Technical Skills Acquired

**Frontend Development:**
- HTML5: Semantic markup, structure, text formatting, lists, links, images, tables, and multi-page site architecture
- CSS3: Styling, layout design, responsive design principles, and CSS frameworks (Bootstrap)
- JavaScript: 
  - Core programming concepts (variables, data types, operators)
  - Functions, methods, and objects
  - Control flow (decisions and loops)
  - Event handling and DOM manipulation
  - Form validation (client-side)
  - jQuery integration

**Backend Development:**
- PHP: 
  - Server-side scripting and PHP syntax
  - Form handling (GET and POST methods)
  - User input processing and validation
  - Session management
  - Password hashing and authentication
- MySQL Database:
  - Database design and schema creation
  - SQL queries (INSERT, SELECT)
  - PHP-MySQL integration using mysqli
  - phpMyAdmin administration

**DevOps & Deployment:**
- AWS EC2 server setup and configuration
- LEMP stack installation (Linux, Nginx, MySQL, PHP)
- SSL/TLS certificate configuration
- Web server security best practices
- Domain configuration and DNS management

**Development Tools:**
- Adobe Dreamweaver / Notepad++ for code editing
- Browser developer tools for debugging
- Firefox Developer Edition for testing
